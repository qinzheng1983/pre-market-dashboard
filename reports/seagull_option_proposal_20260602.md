## 二、海鸥期权方案设计

### 2.1 海鸥期权结构（Seagull Structure）

海鸥期权是一种**三腿期权组合**，通过卖出两个期权来降低/消除买入保护性期权的成本。

**针对我司 Long USD 敞口（担心人民币升值）的标准海鸥结构：**

```
┌─────────────────────────────────────────────────────────────┐
│              USD/CNY 看跌海鸥期权（6个月期限）                  │
├─────────────────────────────────────────────────────────────┤
│  腿1：买入看跌期权（保护性）                                   │
│        执行价 K1 = 6.70（价外约65点，即期6.7645）              │
│        作用：USD/CNY跌破6.70时提供保护                         │
│        成本：支付权利金 ~0.9%（名义本金）                       │
├─────────────────────────────────────────────────────────────┤
│  腿2：卖出更低执行价看跌期权（降低保费）                         │
│        执行价 K2 = 6.60（价外约165点）                         │
│        作用：收取权利金，降低总成本                             │
│        收入：收取权利金 ~0.4%（名义本金）                       │
├─────────────────────────────────────────────────────────────┤
│  腿3：卖出更高执行价看涨期权（降低保费）                         │
│        执行价 K3 = 6.85（价外约85点）                          │
│        作用：收取权利金，进一步降低总成本                       │
│        收入：收取权利金 ~0.5%（名义本金）                       │
├─────────────────────────────────────────────────────────────┤
│  净权利金成本：约 0.0%（接近零成本）                           │
│  名义本金：10亿美元 × 对冲比例（建议30-50%）                   │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 收益结构图示

```
USD/CNY 汇率
    │
7.00├───────────────────────────── 上方收益封顶（K3=6.85）
    │                              卖出看涨生效：美元升值>6.85无额外收益
6.85├───────────────────────────── 卖出看涨执行价
    │     正常敞口区间（无期权影响）
6.76├────────────●────────────── 当前即期汇率
    │              │
6.70├─────────────┼────────────── 买入看跌执行价（K1）
    │              │  保护区间：6.60-6.70有保护
6.60├─────────────┼────────────── 卖出看跌执行价（K2）
    │              │  │
    │              │  │  下方双重保护失效
6.50├─────────────┼──┼────────── USD/CNY<6.60回到裸敞口
    │              │  │
    │  裸敞口      │  │  裸敞口
    └──────────────┴──┴──────────
```

### 2.3 各区间损益分析（以50%对冲比例，5亿美元名义本金为例）

| USD/CNY区间 | 期权组合状态 | 敞口损益 | 期权损益 | 净效果 |
|------------|-------------|---------|---------|--------|
| > 6.85 | 卖出看涨生效 | 美元升值收益 | 收益封顶，无法享受>6.85部分 | 收益受限 |
| 6.70 - 6.85 | 无期权生效 | 正常敞口 | 无 | 正常敞口 |
| 6.60 - 6.70 | 买入看跌生效 | 美元贬值损失 | 部分补偿（6.70-实际汇率） | 部分保护 |
| < 6.60 | 双重看跌生效 | 美元大幅贬值损失 | 买入看跌+卖出看跌抵消，回到裸敞口 | **无保护** |

## 三、利弊分析

### 3.1 优势（Pros）

| 优势 | 详细说明 | 对我司价值 |
|------|---------|-----------|
| **零成本/低成本** | 通过卖出两个期权收取的权利金，基本覆盖买入看跌的成本 | 不占用额外资金，适合当前流动性管理需求 |
| **保留中间区间收益** | 6.70-6.85之间正常敞口，不受影响 | 不对正常经营造成干扰 |
| **部分下行保护** | 6.60-6.70区间有买入看跌保护 | 覆盖温和升值情景（约2.4%贬值幅度） |
| **适合当前市场环境** | 央行连续调升中间价，人民币温和升值趋势 | 与市场预期一致，不逆势操作 |
| **结构灵活** | 执行价、期限、对冲比例可调整 | 可根据敞口变化动态调整 |
| **会计处理友好** | 期权组合可指定为现金流量套期或公允价值套期 | 满足IFRS 9或CAS套期会计要求 |

### 3.2 劣势与风险（Cons）

| 劣势 | 详细说明 | 风险等级 | 对我司影响 |
|------|---------|---------|-----------|
| **极端贬值无保护** | USD/CNY跌破6.60后，双重看跌期权失效，回到裸敞口 | 🔴 **高** | 若人民币大幅升值（如到6.50），10亿USD敞口无保护，损失巨大 |
| **上方收益封顶** | USD/CNY突破6.85后，无法享受美元升值收益 | 🟡 中 | 若美元反转走强，放弃潜在收益 |
| **结构复杂** | 三腿期权，Delta/Gamma管理复杂 | 🟡 中 | 需要专业团队持续监控，或委托银行管理 |
| **流动性风险** | 若汇率在6.60附近震荡，卖出看跌可能被行权 | 🟡 中 | 需要现金准备或提前平仓 |
| **基差风险** | 期权基于中间价或即期，与敞口实际结算价可能存在差异 | 🟡 中 | 对冲效果不完全精确 |
| **机会成本** | 如果选择领子或远期，可能获得更稳健的保护 | 🟡 中 | 海鸥在极端情景下保护不足 |

## 四、情景分析（概率加权）

基于当前市场环境（央行连续调升、美元偏弱、美伊谈判僵持），设定以下情景及概率：

### 4.1 五情景矩阵

| 情景 | USD/CNY走势 | 触发条件 | 概率 | 海鸥期权效果 | 对我司影响 | 净损益（5亿名义） |
|------|------------|---------|------|------------|-----------|-----------------|
| **1. 温和升值** | 6.70 - 6.80 | 央行继续引导升值，但 pace 放缓 | 35% | 无影响（正常敞口） | 资本层面正常折算 | 无额外对冲损益 |
| **2. 大幅升值** | < 6.60 | 中美利差收窄、资本流入、美元系统性走弱 | 20% | **保护失效**（跌破6.60） | 资本层面损失巨大 | 无保护，裸敞口损失 |
| **3. 温和贬值** | 6.80 - 6.85 | 美伊冲突升级、美元避险反弹 | 25% | 无影响（正常敞口） | 资本层面折算收益增加 | 无额外对冲损益 |
| **4. 大幅贬值** | > 6.85 | 美联储鹰派转向、中国央行放松干预 | 15% | **收益封顶** | 放弃>6.85部分收益 | 封顶损失（如到7.00，放弃1500点×5亿） |
| **5. 区间震荡** | 6.58 - 6.62 | 多空交织、政策博弈 | 5% | 双重看跌生效，复杂对冲 | 对冲损益波动 | 取决于具体路径 |

### 4.2 各情景详细展开

#### 情景1：人民币温和升值（6.70-6.80，概率35%）

**市场逻辑**：
- 央行继续通过中间价引导升值，但 pace 放缓（每日调升10-20点）
- 美伊谈判若达成阶段性协议，避险情绪消退，人民币获支撑
- 中国贸易顺差维持高位，结汇需求支撑汇率

**海鸥期权效果**：
- 三个期权腿均不生效
- 期权组合处于"休眠"状态
- 净效果：零成本持有，正常敞口

**对我司影响**：
- 资本层面10亿USD：正常折算，无额外对冲损益
- 海鸥期权实现"零成本保险"目标

**ACTION**：无特别操作，持续监控中间价调整 pace。

---

#### 情景2：人民币大幅升值（< 6.60，概率20%）

**市场逻辑**：
- 中美利差大幅收窄（中国降息+美国加息）
- 美伊冲突迅速缓和，霍尔木兹海峡完全开放，油价暴跌
- 中国资本账户进一步开放，外资大幅流入
- 或者：中美达成重大贸易协议，人民币一次性重估

**海鸥期权效果**：
- 买入看跌6.70生效：在6.60-6.70区间提供部分保护
- 卖出看跌6.60生效：跌破6.60后，双重看跌抵消，回到裸敞口
- **关键风险**：在6.60以下完全无保护

**对我司影响（假设到6.50）**：
- 无对冲保护：10亿USD × (6.7645 - 6.50) = **2.645亿人民币损失**
- 海鸥部分保护：仅保护6.70-6.60区间（约0.10元），即1亿人民币
- **未保护损失**：1.645亿人民币

**ACTION**：若汇率逼近6.65，紧急评估是否追加保护（如买入6.50看跌期权或远期锁定）。

---

#### 情景3：美元温和反弹（6.80-6.85，概率25%）

**市场逻辑**：
- 美伊谈判破裂，霍尔木兹海峡风险重燃
- 美联储释放鹰派信号，美元指数反弹
- 中国央行阶段性放松干预，允许人民币适度贬值

**海鸥期权效果**：
- 三个期权腿均不生效
- 正常享受美元升值收益

**对我司影响**：
- 资本层面10亿USD：折算收益增加
- 海鸥期权无额外成本

**ACTION**：无特别操作，若突破6.85需关注卖出看涨行权风险。

---

#### 情景4：美元大幅升值（> 6.85，概率15%）

**市场逻辑**：
- 美联储重启加息周期，中美利差逆转
- 中国央行大幅降准降息，货币政策宽松
- 全球避险情绪急剧升温，美元流动性危机

**海鸥期权效果**：
- 卖出看涨6.85生效：上方收益封顶
- 无法享受USD/CNY > 6.85的额外收益

**对我司影响（假设到7.00）**：
- 敞口收益：10亿USD × (7.00 - 6.7645) = 2.355亿人民币
- 海鸥封顶：仅享受6.85以下部分，即10亿 × (6.85 - 6.7645) = 0.855亿
- **放弃收益**：1.50亿人民币（机会成本）

**ACTION**：若美元强势确立，评估是否提前平仓海鸥期权，转为远期锁定或买入看涨。

---

#### 情景5：区间震荡（6.58-6.62，概率5%）

**市场逻辑**：
- 多空力量均衡，政策博弈激烈
- 汇率在6.60附近反复震荡

**海鸥期权效果**：
- 买入看跌6.70和卖出看跌6.60同时生效
- Delta对冲复杂，Gamma风险暴露
- 可能产生" Ping-Pong "效应（反复行权/被行权）

**对我司影响**：
- 对冲损益波动，难以预测
- 可能产生意外的现金流入/流出

**ACTION**：若进入震荡区间，建议与交易对手协商调整执行价或提前平仓。

### 4.3 概率加权期望值

| 情景 | 概率 | 海鸥期权效果 | 期望贡献 |
|------|------|------------|---------|

## 五、对比方案：海鸥 vs 其他工具

### 5.1 工具对比矩阵

| 工具 | 成本 | 下方保护 | 上方收益 | 适合情景 | 对我司适用度 |
|------|------|---------|---------|---------|------------|
| **海鸥期权** | 零/低 | 部分（6.60-6.70） | 封顶（6.85） | 温和升值 | ⭐⭐⭐⭐ |
| **领子期权（Collar）** | 低 | 完整（无下限） | 封顶 | 区间波动 | ⭐⭐⭐⭐⭐ |
| **买入看跌期权** | 高（~1.5%） | 完整 | 完整 | 强烈贬值预期 | ⭐⭐⭐⭐ |
| **远期锁定** | 零（但贴水） | 完整 | 完整（已锁定） | 确定性需求 | ⭐⭐⭐⭐ |
| **比率远期** | 零/负 | 非对称 | 非对称 | 大额敞口 | ⭐⭐⭐⭐ |

### 5.2 推荐组合方案

**不建议单一工具覆盖全部敞口**。基于我司约10亿美元Long USD敞口，推荐分层对冲：

```
┌─────────────────────────────────────────────────────────────┐

│      作用：低成本覆盖温和升值情景                              │
│      工具：6个月海鸥（K1=6.70, K2=6.60, K3=6.85）             │
│      成本：接近零                                             │

│  综合效果：                                                   │
│  - 90%敞口有保护                                              │
│  - 极端升值（<6.60）：领子+远期提供完整保护                    │
│  - 极端贬值（>6.90）：10%裸敞口+领子看涨封顶保留部分收益        │
```
