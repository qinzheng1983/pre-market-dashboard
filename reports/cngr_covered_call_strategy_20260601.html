中伟新材 USD/CNY Covered Call 策略方案

策略类型：持有美元多头 + 卖出美元看涨期权 | 生成时间：2026-06-01 06:49 CST
一、敞口诊断：为什么选中伟做 Covered Call
敞口维度方向金额量级汇率敏感性
北美出口应收（特斯拉等）Long USD最大USD升值→收入增加
资本层面（子公司美元负债）Long USD大USD升值→负债折算减少
香港购汇（印尼子公司支付）Short USD中USD升值→购汇成本上升
净敞口判断：北美出口 + 资本层面 Long USD 显著大于香港购汇 Short USD → 净美元多头（Net Long USD）
核心矛盾：2025年汇兑损益激增45%至12.69亿元，验证了一个事实——人民币升值时中伟新材受损远大于受益。净美元多头暴露在人民币单边升值风险中，需要结构性对冲。
为什么不是 Covered Put：Covered Put = 持有美元空头 + 卖出美元看跌期权，适合"有美元应付、赌美元不跌"。中伟的美元空头（香港购汇）规模较小，且当前美元在走弱，卖Put等于赌美元反弹，与趋势相悖。Covered Call 才是匹配中伟敞口方向的正确策略。

三、策略设计：Covered Call 具体参数
策略结构 = 持有美元应收（标的）+ 卖出 USD/CNY Call（美元看涨期权）
盈亏本质 = 用"放弃美元大幅升值收益"换取"期权费收入"，补贴人民币贬值损失
参数设定值逻辑
标的敞口USD 100M（举例，按实际敞口调整）北美出口应收 + 资本层面净敞口
期权类型USD Call / CNY Put卖出"美元升值"的权利
执行价 K6.8500高于即期6.7685约1.2%，覆盖中间价6.8176上方的反弹空间
到期日3个月（2026年8月底）匹配应收账期60-90天 + Q2-Q3过渡
期权费（估算）年化 2.0-2.5%（3个月约0.5-0.6%）当前隐含波动率中等，DXY偏弱环境下Call费不高
名义本金USD 50M（对冲覆盖50%敞口）保留50%裸敞口应对美元反弹，避免过度保守
交易对手中资大行场外期权台（中银/工银/建银）场外定制，执行价和到期可灵活调整
⚠️ 参数调整建议：如果实际净美元敞口远大于100M，可按比例放大名义本金。如果敞口小于50M，建议全额覆盖（100% Covered Call），因为小额敞口下"保留裸敞口"的管理成本高于收益。
四、盈亏分析：三种情景推演
到期时 USD/CNY = S
• 美元应收价值 = S × 50M USD（按对冲部分计算）
• Call 期权盈亏 = -max(0, S - 6.85) × 50M USD（被行权时交付美元）
• 期权费收入 = 约 50M × 6.77 × 0.55% ≈ CNY 186万（一次性收取）
• 综合效果 = 美元应收价值 + 期权费 - Call 履约成本
情景到期 USD/CNY美元应收（50M）Call 履约期权费综合人民币收入vs 裸敞口对比
A. 人民币大幅升值6.603.30亿0（失效）+186万3.3186亿比裸敞口多186万（期权费补贴）
B. 区间震荡6.753.375亿0（失效）+186万3.3936亿比裸敞口多186万
C. 美元反弹但 capped6.903.45亿-250万+186万3.4436亿比裸敞口少64万（收益被cap）
D. 美元大涨7.103.55亿-1,250万+186万3.4436亿比裸敞口少1,064万（上端收益封顶）
核心结论：
1. 情景A/B（人民币升值/横盘）：Covered Call 最优。期权费直接补贴贬值损失，裸敞口无保护时亏多少，Covered Call亏得少186万。
2. 情景C（温和反弹）：Covered Call 略差。美元涨到6.90，裸敞口赚更多，但差距仅64万（可接受）。
3. 情景D（大涨）：Covered Call 明显吃亏。但当前DXY 98.91 + 美伊和平预期 + 美联储偏鸽，美元大涨到7.10的概率较低（<15%）。
五、为什么不选 Covered Put（策略排除逻辑）
维度Covered CallCovered Put
适用敞口美元多头（中伟匹配）美元空头（中伟不匹配）
当前操作持有美元 + 卖 USD Call做空美元 + 卖 USD Put
对中伟意味着应收美元贬值时，期权费补贴损失购汇成本上升时，期权费补贴成本
中伟敞口规模大（北美出口+资本）小（香港购汇）
当前趋势适配美元走弱 → Call 不易被行权，赚时间价值美元走弱 → Put 容易被行权，被迫高价买入美元
风险方向上端收益封顶（可接受）下端敞口无限（裸卖Put风险极大）
排除 Covered Put 的致命理由：如果选 Covered Put，等于在美元已经走弱的时候"赌美元不跌"，一旦USD/CNY继续跌到6.60，Put被行权意味着中伟被迫以6.85（或更高）买入美元——而即期只有6.60，直接亏损250点/美元。这是方向性错误，敞口错配 + 趋势错配。

6到期前30天评估展期或调整执行价2026年7月底财资部
替代方案（更保守）：如果担心美元反弹超预期，可将 Covered Call 升级为 Collar（领子期权） = 卖出 Call @6.85 + 买入 Put @6.65。这样下端也有保护，但期权费净支出接近零（甚至略有收入）。代价是下端保护消耗了部分期权费收入。
替代方案（更激进）：如果认为美元反弹概率极低，可卖出 虚值更深的 Call @6.90，期权费更低但上端空间更大。适合"强烈看空美元"的情景。
七、风险边界与止损纪律
风险类型触发条件应对措施
美元意外大涨USD/CNY突破6.90评估提前平仓Call（可能亏损），或接受收益封顶
波动率飙升地缘冲突升级（美伊谈判破裂）Call被行权概率大增，但美元应收同步增值，净损失有限
交易对手风险银行信用恶化只选系统重要性银行（中农工建），要求CSA抵押品
流动性风险到期前需要提前平仓场外期权流动性差，尽量持有到期；如需平仓，折价可能10-20%
八、一句话总结
中伟新材是净美元多头，当前美元偏弱、人民币持续升值。Covered Call = 持有美元应收 + 卖出 USD/CNY Call @6.85，收取期权费补贴贬值损失，放弃小概率的美元大涨收益。这是敞口方向正确、趋势适配、盈亏比合理的策略选择。
