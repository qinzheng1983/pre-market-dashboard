# 盘前简报数据源规范

## 数据源优先级

### USD/CNY中间价（最高优先级，唯一铁律）
**ONLY认以下渠道**：
1. 中国货币网 chinamoney.com.cn / chinamoney.org.cn
2. 中国人民银行官网 pbc.gov.cn
3. 新浪财经/财联社等正规媒体转载的央行公告

**绝对禁止使用**：
- chl.cn（汇率网）
- 查汇率等聚合站
- 任何仅写"今日中间价"无明确央行来源的页面

**校验流程**：
```
1. 搜索 "YYYY年MM月DD日 中国外汇交易中心 人民币汇率中间价公告"
2. 找到央行公告 → 确认公告日期 = 报告日期
3. 提取中间价数字
4. 用 chinamoney.com.cn/chinese/bkccpr/ 二次确认
5. 标注来源时写："央行授权公告 + 新浪财经/财联社"
```

### USD/CNY离岸/在岸价
**必用双源**：
- Investing.com（主源）
- 东方财富 / 新浪财经（验证源）

**异常拦截**：
- 离岸与在岸价差 > 50点时，必须第三源验证，否则标记"待确认"
- 禁止使用"我查网""实时汇率网"等聚合站

### 美股收盘
- 新浪财经 finance.sina.com.cn
- cnfol.com
- 东方财富

### 布伦特原油
- Investing.com 历史数据表：cn.investing.com/commodities/brent-oil-historical-data
- 必须确认日期列是当日，不是前一日
- 金十数据 / 新浪财经 作为验证

### LME有色金属
- LME官方 / Westmetall
- 财联社 / SMM
- 差异 > $50需第三源确认

### 碳酸锂
- SMM上海有色网（主源）
- Mysteel / 生意社（验证）

### 现货黄金
- 格隆汇 / 金价查询网
- Investing.com

### 美元指数DXY
- Investing.com

### 恒生指数
- 新浪财经 / 东方财富

### 地缘风险
- 搜索 "霍尔木兹海峡 以色列 伊朗 最新"
- 路透社 / 财联社 / 央视新闻

## 数据时效声明格式

融入数据来源板块底部：
```
| 数据类别 | 最新日期 | 更新频率 | 状态 |
|---------|---------|---------|------|
| USD/CNY中间价 | YYYY-MM-DD | 日度（9:15） | ✅ |
```

## 交叉验证标准

| 数据类型 | 最低源数 | 差异阈值 | 处理 |
|---------|---------|---------|------|
| 中间价 | 2（央行+货币网） | 无差异要求 | 必须一致 |
| 离岸价 | 2 | >50点 | 第三源或标记待确认 |
| LME镍 | 2 | >$50 | 第三源确认 |
| 原油 | 2 | >$0.5 | 第三源确认 |
| 碳酸锂 | 2 | >500元 | 第三源确认 |
