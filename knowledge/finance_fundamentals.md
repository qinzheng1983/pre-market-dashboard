## 一、货币性项目 vs 非货币性项目（CPA会计核心）

### 定义
- **货币性项目**：企业持有的货币资金、将以固定或可确定金额收取的资产/偿付的负债
  - 资产：现金、银行存款、应收账款、应收票据
  - 负债：应付账款、短期借款、长期借款、应付债券
- **非货币性项目**：除货币性项目以外的项目，金额不固定，随汇率波动变化
  - 存货、固定资产、无形资产、实收资本、留存收益

### 汇兑损益处理规则
```
货币性项目：资产负债表日/结算日 → 按即期汇率折算 → 差额计入财务费用（汇兑损益）
非货币性项目：历史成本计量 → 交易日汇率入账 → 后续不调整
```

### 关键记忆口诀
> **"资产在外币升值时收益，负债在外币升值时损失"**
> - 持有美元资产（Long USD）→ 美元升值 → 折算人民币增加 → **收益**
> - 承担美元负债（Short USD）→ 美元升值 → 偿还成本增加 → **损失**

## 二、汇率敞口方向判断（绝不再混淆）

### 敞口方向定义
| 术语 | 含义 | 外汇变动影响 |
|------|------|-------------|
| **Long USD** | 持有美元资产/未来收到美元 | USD升值→有利 |
| **Short USD** | 承担美元负债/未来支付美元 | USD升值→不利 |

### 敞口判断检查清单
生成报告前必须逐项确认：
1. [ ] 应收账款/应收票据 → **Long**（未来收外币）
2. [ ] 应付账款/应付票据 → **Short**（未来付外币）
3. [ ] 外币借款/债券 → **Short**（未来还外币）
4. [ ] 外币存款/现金 → **Long**（持有外币）
5. [ ] 预收账款 → **Long**（未来收外币）
6. [ ] 预付账款 → **Short**（已付外币，但货物非货币性）

### 净敞口计算公式
```
净敞口 = Σ外币资产 - Σ外币负债
       = 所有Long敞口 - 所有Short敞口

结果 > 0 → 净Long → 外币升值有利
结果 < 0 → 净Short → 外币升值不利
```

### ⚠️ 血泪教训（2026-04-28）
**错误**：将印尼子公司美元负债判断为 Long USD
**正确**：美元负债 = 未来要偿还美元 = **Short USD**
**后果**：净敞口计算从 +$2,000万（Long）变成 -$6,000万（Short），结论完全反转
**根因**：混淆了"持有子公司股权"（股权投资，非货币性）和"子公司欠的美元债"（货币性负债）

## 三、功能货币与报表折算（CFA FRA）

### 三种外币报表折算方法
| 方法 | 适用场景 | 资产负债表 | 利润表 | 折算差额 |
|------|---------|-----------|--------|---------|
| **现行汇率法** | 功能货币 ≠ 记账货币，独立经营 | 所有资产/负债按期末汇率 | 按平均汇率 | 其他综合收益（OCI） |
| **时态法** | 功能货币 = 母公司货币，经营不独立 | 货币性项目按期末汇率，非货币性按历史汇率 | 与资产负债表匹配 | 利润表 |
| **超通货膨胀经济** | 通胀率>100%（3年累计） | 重新计量 | 重新计量 | 利润表 |

### 功能货币判定因素（CFA）
1. 销售市场定价货币
2. 人工/材料/费用支付货币
3. 融资活动货币
4. 经营成果保存货币

### 中伟新材印尼子公司
- 印尼基地销售：出口韩国、日本（USD结算）+ 本地（IDR结算）
- 成本支付：IDR为主（当地人工、矿产）
- 融资：USD负债（$4,000万）
- **判断**：功能货币可能是IDR或USD，需具体分析。若为IDR，报表折算用现行汇率法；若为USD，可能涉及时态法。

## 四、对冲会计（Hedge Accounting）

### 三种对冲类型
| 类型 | 对冲对象 | 会计处理 |
|------|---------|---------|
| **公允价值对冲** | 已确认资产/负债或承诺的公允价值变动 | 对冲工具+被对冲项目损益均计入当期 |
| **现金流量对冲** | 未来极可能发生的预期交易现金流 | 有效部分→OCI，无效部分→当期损益 |
| **净投资对冲** | 境外经营净投资的外汇风险 | 有效部分→OCI，处置时转入损益 |

### 中伟新材适用分析
- **USD/CNY中间价波动** → 影响的是已确认的应收/应付/负债 → **公允价值对冲**
- **预期未来出口收款** → 未来现金流 → **现金流量对冲**
- **印尼子公司净投资** → 境外经营净投资 → **净投资对冲**

## 五、衍生工具基础（CFA Derivatives）

### 远期合约（Forward）
```
远期多头（Long Forward）：约定未来以K价格买入标的 → 标的涨价有利
远期空头（Short Forward）：约定未来以K价格卖出标的 → 标的跌价有利

外汇远期应用：
- 锁定未来购汇成本（担心USD升值）→ 买入USD/CNY远期多头
- 锁定未来结汇收入（担心USD贬值）→ 卖出USD/CNY远期空头
```

### 期权基础
| 期权类型 | 权利 | 支付 | 适用场景 |
|---------|------|------|---------|
| 看涨期权（Call） | 以K买入标的 | 期权费（Premium） | 预期涨价，锁定买入成本 |
| 看跌期权（Put） | 以K卖出标的 | 期权费（Premium） | 预期跌价，锁定卖出收入 |

### 外汇期权应用（中伟新材）
```
净Short USD $6,000万/月 → 担心USD升值 → 需要"买入美元的权利"
→ 买入USD/CNY看涨期权（Call Option）
→ 支付期权费，获得以约定汇率买入美元的权利
→ 若USD大幅升值→行权，锁定成本
→ 若USD贬值→放弃行权，享受市场低价
```

## 六、投行分析师视角：企业汇率风险管理框架

### 1. 敞口识别
- 交易敞口（Transaction Exposure）：已确认的应收/应付
- 折算敞口（Translation Exposure）：报表折算风险
- 经济敞口（Economic Exposure）：未来现金流长期竞争力

### 2. 对冲策略选择矩阵
| 敞口类型 | 时间维度 | 推荐工具 | 对冲比例 |
|---------|---------|---------|---------|
| 交易敞口 | <1年 | 远期、期权 | 60-80% |
| 交易敞口 | 1-3年 | 远期、货币互换 | 40-60% |
| 折算敞口 | 持续 | 净投资对冲 | 视净投资规模 |
| 经济敞口 | 长期 | 自然对冲（本地化生产） | N/A |

### 3. 自然对冲策略
- **货币匹配**：在USD收入市场用USD融资（借美元还美元）
- **本地化生产**：韩国基地用KRW成本生产，本地销售收KRW
- **币种多元化**：IDR收入用于IDR成本，减少IDR/CNY敞口
