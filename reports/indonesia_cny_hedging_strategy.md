# 印尼子公司 CNY/USD 汇率风险套期保值策略方案

**编制日期：2026年5月10日**  
**适用实体：印尼子公司（记账本位币：USD）**  
**敞口规模：月均约等值2亿美元的人民币货款流入**  
**策略目标：锁定CNY→USD兑换汇率，消除汇率波动对报表及现金回流的不确定性**

---

## 一、执行摘要

| 维度 | 结论 |
|------|------|
| **敞口方向** | **Long CNY / Short USD**（持有人民币应收账款/现金，折算成美元时面临人民币贬值风险） |
| **敞口规模** | 月均约 CNY 13.6亿元（按USD/CNY 6.80估算），年敞口约 CNY 163亿元 |
| **核心风险** | CNY兑美元贬值 → USD报表收入/利润下降；若需汇回母公司，实际可兑换USD减少 |
| **最优工具** | **CNY/USD FX Forward（远期外汇合约）** via 中国银行雅加达分行 |
| **对冲比例建议** | 第一年50%→第二年70%，第三年根据自然对冲成熟度动态调整 |
| **策略成本** | 远期点（Forward Points）约为USD/CNY即期的 **-1.5%至-2.0%（年化贴水）**，实际对冲成本接近零（利率平价覆盖） |
| **合规路径** | 以月度出口销售合同为Underlying Document，符合BI实需原则 |

> **一句话策略**：通过中国银行雅加达分行，以逐月出口合同为实需依据，滚动签署3/6/12个月CNY/USD远期卖出合约，将月均2亿美元等值人民币的汇率波动锁定在可预测区间。

## 二、敞口深度分析

### 2.1 敞口方向判定

```
子公司记账本位币：USD
子公司每月流入：CNY（等值约2亿USD）

会计处理：
  收到CNY货款 → 确认CNY应收账款/银行存款
  月末报表折算 → CNY余额 × USD/CNY期末汇率 → 计入USD报表

敞口性质：
  - 子公司是CNY资产的持有者（Long CNY）
  - 但报表/经营需要用USD计量（Short USD隐含敞口）
  - 若CNY贬值：同样的CNY余额兑换成USD更少 → 报表损失
  
对冲方向：卖出CNY / 买入USD（或等效地：锁定未来CNY→USD的兑换率）
```

### 2.2 敞口规模量化

| 项目 | 金额 | 说明 |
|------|------|------|
| 月均CNY流入 | ~CNY 13.6亿元 | 按USD/CNY 6.80折算2亿USD |
| 年均CNY流入 | ~CNY 163亿元 | 12个月累计 |
| 汇率敏感度 | ±1% CNY贬值 → ±200万美元/月 | 月均敞口对汇率敏感度 |
| 年度汇率敏感度 | ±1% CNY贬值 → ±2,400万美元/年 | 全年敞口对汇率敏感度 |

**压力测试（年度）**：

| 情景 | USD/CNY变动 | 年度汇兑影响（vs即期6.80） |
|------|-------------|---------------------------|
| 基准 | 维持6.80 | 0 |
| CNY小幅贬值 | 升至7.00（+2.9%） | 年度损失约 **6,960万美元** |
| CNY中度贬值 | 升至7.20（+5.9%） | 年度损失约 **1.42亿美元** |
| CNY大幅贬值 | 升至7.50（+10.3%） | 年度损失约 **2.48亿美元** |
| CNY升值 | 降至6.50（-4.4%） | 年度收益约 **1.06亿美元**（未对冲时的"意外之财"） |

> **关键判断**：子公司是净出口商收CNY，不是投机者。一年2.4亿美元的潜在波动对任何企业都是不可承受之重。必须对冲。

## 三、印尼外汇监管与可用工具

### 3.1 印尼外汇监管框架（Bank Indonesia）

| 法规 | 核心要求 | 对本策略的影响 |
|------|---------|--------------|
| **PBI No. 6/2024 + PADG No. 11/2024** | 外汇衍生品交易须有Underlying Transaction（实需原则） | ✅ 以月度出口合同为实需依据，完全合规 |
| **PADG No. 7 of 2026（2026年4月生效）** | 提高无基础文件交易门槛：DNDF/Forward卖出USD≥1000万美元/笔；Swap≥1000万美元/笔 | ✅ 月均2亿美元远超门槛，且有出口合同作为基础文件，不受此限制 |
| **BI 现金购汇限制（2026年5月收紧）** | 个人无基础文件现金购汇上限降至USD 25,000/人/月 | ⚠️ 对企业衍生品交易无直接影响，但若涉及员工购汇需注意 |
| **DHE SDA规则** | 自然资源出口须将外汇收益存入印尼特别账户≥12个月 | ❌ 不适用（非自然资源出口） |
| **实需原则（Underlying Document）** | 衍生品名义本金≤基础交易金额；期限≤基础交易期限 | ✅ 远期合约金额=月度销售额，期限≤应收账款账期 |

### 3.3 关键发现：中国银行雅加达分行的独特优势

> **中国银行雅加达分行是印尼最大的人民币金融服务银行，也是中印尼双边本币结算（LCS）指定的ACCD银行。**

- 2023年占印尼ACCD银行交易量的 **81.3%**
- 提供：**CNY/USD 即期、远期、掉期**
- 也是印尼最大人民币现钞供应商
- 2023年成为印尼首个人民币跨境支付系统（CIPS）直参行
- **对中资企业而言，这是印尼本地唯一能直接做CNY/USD远期的银行**

## 四、推荐策略方案：CNY/USD远期锁汇（核心方案）

### 4.1 策略架构

```
┌─────────────────────────────────────────────────────────┐
│                    印尼子公司                             │
│              （记账本位币：USD）                          │
│                                                         │
│   每月收到CNY货款（等值2亿USD）                           │
│            ↓                                            │
│   ┌─────────────────────────────┐                     │
│   │  50%-80% 做远期锁汇          │                     │
│   │  • 3个月远期：滚动签署        │                     │
│   │  • 6个月远期：覆盖中期        │                     │
│   │  • 12个月远期：锁定年度预算   │                     │
│   └─────────────────────────────┘                     │
│            ↓                                            │
│   剩余20%-50%：自然对冲/即期兑换                          │
│            ↓                                            │
│   USD现金/汇回母公司                                     │
└─────────────────────────────────────────────────────────┘
```

### 4.2 对冲比例动态调整路径

| 阶段 | 时间 | 对冲比例 | 理由 |
|------|------|---------|------|
| **启动期** | 第1-6个月 | **50%** | 建立对冲习惯，测试银行流程，保留灵活性观察市场 |
| **成熟期** | 第7-18个月 | **70%** | 流程顺畅后提高覆盖，仅留30%博弈CNY升值 |
| **优化期** | 第19个月起 | **50%-80%动态** | 根据自然对冲成熟度（是否有CNY支出）调整 |

### 4.3 期限结构设计（梯形/滚动策略）

> **核心逻辑**：不把所有敞口押在单一期限，而是分散在3/6/12个月，实现滚动覆盖。

**月度执行示例（以6月收到的CNY货款为例）**：

| 货款月份 | CNY金额（估算） | 3个月远期（9月交割） | 6个月远期（12月交割） | 12个月远期（次年6月交割） |
|----------|---------------|---------------------|----------------------|-------------------------|
| 2026年6月 | CNY 13.6亿 | 锁定30% = CNY 4.08亿 | 锁定40% = CNY 5.44亿 | 锁定30% = CNY 4.08亿 |
| 2026年7月 | CNY 13.6亿 | 锁定30% = CNY 4.08亿 | 锁定40% = CNY 5.44亿 | 锁定30% = CNY 4.08亿 |
| ... | ... | ... | ... | ... |

**远期合约滚动机制**：
- 每笔远期到期时，子公司提供CNY，银行提供USD，完成交割
- 若到期时子公司暂无足够CNY（比如客户延迟付款），可提前与银行协商 **展期（Roll）** 或 **平仓后重新开仓**
- 建议与银行约定 **差额交割（Net Settlement）** 选项：若到期汇率优于锁定汇率，银行支付差额；若劣于锁定汇率，子公司支付差额——无需全额本金交换

### 4.4 远期定价与成本分析（基于当前市场）

**当前市场参数（2026年5月）**：
- USD/CNY即期：~6.80
- 中国1年期LPR：~3.1%
- 美联储有效利率：~4.25-4.50%
- 中美利差：约 **+120至+140bp（USD高）**

**利率平价下的远期汇率估算**：

根据利率平价公式（简化）：
```
F = S × (1 + r_CNY × T) / (1 + r_USD × T)
```

| 期限 | 估算远期USD/CNY | vs即期 | 年化贴水 |
|------|-----------------|--------|---------|
| 3个月 | ~6.77 | -0.03 | -1.8% |
| 6个月 | ~6.74 | -0.06 | -1.8% |
| 12个月 | ~6.68 | -0.12 | -1.8% |

> **关键洞察**：由于USD利率高于CNY，USD/CNY远期呈现 **贴水（Discount）**。这意味着远期锁定的USD/CNY汇率比即期更低——对持有CNY需要换USD的子公司而言，远期可以多换约1.8%的USD。
> 
> 但这1.8%不是"免费午餐"，而是利率平价的自然结果。如果不做远期、即期换USD后存美元，同样能获得约1.8%的利差收益（扣除信用风险后）。

**实际对冲成本**：
- 远期买卖价差：约 **50-100点（0.5%-0.7%年化）**
- 银行手续费/佣金：约 **0.1%-0.3%名义本金**
- **综合对冲成本：约0.6%-1.0%年化**

## 五、备选方案

### 方案B：自然对冲（零成本，但有限）

**前提条件**：子公司同时有从中国采购原材料/设备的CNY支出。

| 场景 | 操作 | 效果 |
|------|------|------|
| 每月CNY收入 = CNY支出 | 直接匹配收付 | 完全自然对冲，零成本 |
| 每月CNY收入 > CNY支出 | 用收入覆盖支出，剩余做远期 | 部分自然对冲+部分金融对冲 |
| 无CNY支出 | 不适用 | 必须依赖金融对冲 |

> **建议**：即使启动金融对冲，也应同步排查自然对冲可能性。每1亿元CNY的自然对冲，每年可节省约60-100万元的对冲成本。

### 方案C：交叉对冲（CNY/IDR + IDR/USD）

**适用场景**：无法通过中资银行直接做CNY/USD远期，或希望分散交易对手风险。

```
Step 1: 与本地银行（如BCA/Mandiri/Permata）签CNY/IDR远期
        → 卖出CNY，买入IDR
Step 2: 同时签IDR/USD远期（DNDF或常规Forward）
        → 卖出IDR，买入USD
        
净效果：CNY → IDR → USD，实现CNY换USD
```

| 维度 | 评估 |
|------|------|
| 可用性 | ✅ 本地大行均可提供 |
| 基差风险 | ⚠️ CNY/IDR与IDR/USD的远期点可能不完全匹配，产生额外成本 |
| 操作复杂度 | ⚠️ 需要管理两笔合约、两个交割日、两个交易对手 |
| 成本 | 比直接CNY/USD高约 **0.3%-0.5%**（基差+双倍买卖价差） |
| 推荐度 | 仅作为B计划 |

## 六、合规操作路径（印尼本地）

### 6.1 开户与签约流程

```
Step 1: 在Bank of China Jakarta Branch开立企业账户
        └─ 已有账户则跳过
        
Step 2: 签署ISDA主协议（或印尼本地衍生品协议）
        └─ 中行雅加达分行应支持标准化ISDA
        
Step 3: 申请衍生品交易额度
        └─ 提供母公司担保或信用证（如需要）
        
Step 4: 每月提交Underlying Document
        └─ 出口销售合同/发票副本
        └─ 合同金额 ≥ 远期合约名义本金
        └─ 合同期限 ≥ 远期合约期限
        
Step 5: 执行远期交易
        └─ 通过电话/网银/交易专员下单
        └─ 确认交易后收到交易确认书（Confirmation）
```

### 6.2 实需原则（Underlying Document）清单

| 文件类型 | 用途 | 获取难度 |
|----------|------|---------|
| 出口销售合同（Sales Contract） | 证明CNY应收账款的商业基础 | 已有 |
| 商业发票（Commercial Invoice） | 证明交易金额和币种 | 已有 |
| 装箱单/提单（BOL） | 辅助证明货物已发运 | 已有 |
| 付款通知/应收账款确认 | 证明具体应收账款金额和到期日 | 内部生成 |

> **关键合规点**：
> - 每笔远期合约的名义本金 ≤ 对应出口合同的CNY金额
> - 远期交割日 ≤ 应收账款预计到账日
> - 保留完整的文档链备查（BI有权要求检查）

### 6.3 会计处理（印尼本地GAAP + IFRS协调）

| 交易阶段 | 会计分录（简化） | 报表影响 |
|----------|---------------|---------|
| 远期开仓 | 表外披露（或有事项） | 无直接影响 |
| 月末估值 | 按公允价值计量：远期价值变动计入P&L（若未指定Cash Flow Hedge） | 波动增加 |
| 指定CFH后 | 有效部分计入OCI，无效部分计入P&L | 平滑P&L |
| 到期交割 | 按锁定汇率换汇，与即期汇率差异实现 | 锁定效果确认 |

> **强烈建议**：向审计师咨询是否可以采用 **Cash Flow Hedge Accounting**，将远期有效部分的公允价值变动计入 **其他综合收益（OCI）**，而非直接冲击利润表。这能大幅平滑报表波动。

## 七、风险监控与调整机制

### 7.1 监控仪表盘

| 指标 | 频率 | 阈值 | 触发动作 |
|------|------|------|---------|
| USD/CNY即期汇率 | 日 | 突破7.00或跌破6.50 | 评估是否需要调整对冲比例 |
| 对冲覆盖率 | 月 | <40%或>90% | 检查执行偏差 |
| 远期合约到期分布 | 月 | 集中度>50%在单月 | 调整期限结构 |
| 对手方信用风险 | 季 | 银行评级下调 | 考虑更换或分散交易对手 |
| 基差风险（若交叉对冲） | 周 | CNY/IDR+IDR/USD vs CNY/USD价差>100点 | 评估直接对冲替代方案 |

### 7.2 策略调整触发器

| 市场变化 | 策略调整 |
|----------|---------|
| CNY进入明确升值通道（如USD/CNY跌破6.50） | 降低对冲比例至30%，保留更多上行收益 |
| CNY进入明确贬值通道（如USD/CNY突破7.20） | 提高对冲比例至80-90%，甚至考虑Option Collar |
| 中美利差大幅收窄（如中国加息或美联储大幅降息） | 远期贴水缩小，对冲成本下降，可提高对冲比例 |
| 印尼盾剧烈波动（IDR/USD突破18,000） | 若采用交叉对冲，需增加基差监控频率 |
| 获得自然对冲来源（新增CNY采购） | 按采购金额比例降低金融对冲额度 |

