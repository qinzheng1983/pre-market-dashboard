# 财资日报生成器 v3.0

**定制化财资日报 - 覆盖货币政策、新能源、金属大宗**

---

## 覆盖范围

| 模块 | 内容 | 优先级 |
|:-----|:-----|:------:|
| **一、中国货币政策** | 央行操作、市场利率、政策动态 | ⭐⭐⭐⭐⭐ |
| **二、新能源产业** | 动力电池、储能、新能源汽车、新业态 | ⭐⭐⭐⭐⭐ |
| **三、金属与大宗** | 镍铜铝等有色、黑色、贵金属、能源 | ⭐⭐⭐⭐⭐ |
| **四、机构观点** | 货币政策、新能源、大宗的机构研判 | ⭐⭐⭐⭐ |
| **五、行业风险预警** | 风险矩阵、关注事项、对冲建议 | ⭐⭐⭐⭐⭐ |

## 模板结构

### 一、中国货币政策 📊
1. **央行公开市场操作** - 逆回购、MLF、降准降息
2. **货币市场利率** - DR007、SHIBOR、国债收益率、汇率
3. **货币政策动态** - 政策要闻、市场解读、前瞻预判

### 二、新能源产业 🔋
1. **动力电池与储能** - 装车量、招标规模、材料价格
2. **新能源汽车** - 销量、渗透率、出口、厂商动态
3. **新业态与模式** - 换电、光储充、虚拟电厂、氢能
4. **产业链价格追踪** - 碳酸锂、氢氧化锂、电解钴、硫酸镍、石墨

### 三、金属与大宗商品 ⛏️
1. **有色金属（镍为代表）** - 镍、铜、铝、锌、钴的价格与库存
2. **黑色金属** - 铁矿石、螺纹钢、焦煤焦炭
3. **贵金属与能源** - 黄金、白银、原油、天然气
4. **农产品与软商品** - 大豆、棕榈油、棉花

### 四、机构观点汇总 🏦
1. **货币政策观点** - 机构利率预测、汇率预测
2. **新能源行业观点** - 评级、目标价、关键逻辑
3. **大宗商品观点** - 品种观点、目标价位、风险提示

### 五、行业风险预警 ⚠️
1. **风险矩阵** - 政策/市场/供应链/价格/汇率风险
2. **重点关注事项** - 短期/中期/长期关注点
3. **对冲与操作建议** - 汇率/商品价格/利率风险对冲

## 使用方法

### 生成日报
```bash
# 生成完整日报
python3 skills/finance-daily-report/scripts/generate_report_v3.py

# 生成演示版（使用模拟数据）
python3 skills/finance-daily-report/scripts/generate_report_v3_demo.py
```

### 发送日报到邮箱
```bash
python3 skills/email-reporter/scripts/email_reporter.py send \
  --to 13911658378@139.com \
  --subject "财资日报 - $(date +%Y%m%d)" \
  --content "$(cat report.md)"
```
