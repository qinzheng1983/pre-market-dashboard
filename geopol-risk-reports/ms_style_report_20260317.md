## 传导机制: 地缘冲突 → 油价 → 汇率

```
地缘冲突升级 → 霍尔木兹海峡关闭 → 原油供应中断风险
     ↓
战争险保费暴涨 → 船东停航/绕航 → 有效运力骤降
     ↓
油价飙升($67→$103) → 中国进口成本大增 → 贸易顺差收窄
     ↓
资本外流压力 → 人民币贬值压力 → USD/CNY上行
```

**关键传导系数**:
- 油价每涨 $10 → USD/CNY 约 +0.08-0.12
- 海峡关闭每持续1周 → 风险溢价 +5%
