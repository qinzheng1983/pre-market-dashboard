## 二、中伟新材经营质量深度分析

### 2.1 经营活动现金流：⚠️ 显著恶化

**核心问题：经营现金流大幅下滑65.2%**

从39.43亿元骤降至13.74亿元，主要原因：

| 因素 | 影响分析 |
|------|----------|
| **存货激增** | 存货增加49.81亿元（附注显示从98.26亿增至146.64亿），占压大量资金 |
| **应收项目增加** | 经营性应收项目增加31.43亿元 |
| **应付项目对冲** | 经营性应付项目增加40.91亿元，部分抵消上述影响 |

**CFA视角解读：**
- **现金流/净利润比率** = 13.74亿 / 16.11亿 = 0.85，低于健康水平（通常>1.0为佳）
- **盈利质量存疑**：净利润16.11亿元，但经营现金流仅13.74亿元，说明利润中含有较多非现金项目
- **营运资本管理压力**：存货周转天数增加，应收账款回收放缓

### 2.2 投资活动现金流：⚠️ 持续大额资本支出

**投资净流出53.97亿元**，虽然较2024年的99.72亿元有所改善，但仍处于高位：

| 投资活动明细 | 2025年金额 | 说明 |
|--------------|------------|------|
| 购建固定资产等 | 39.85亿元 | 印尼镍项目、磷酸铁项目等 |
| 投资支付现金 | 99.88亿元 | 理财产品及股权投资 |
| 取得子公司 | 14.08亿元 | 收购Wintru、Solaroz等矿产公司 |
| 资金拆借净流出 | 10.81亿元 | 关联方资金拆借 |

**风险提示**：
- 印尼镍矿项目资本密集，投资回报周期长
- 2025年完成H股上市融资（31.46亿元），部分用于补充投资资金
- 投资回报率（ROIC）需持续跟踪验证

### 2.3 筹资活动现金流：⚠️ 融资依赖度高

| 筹资来源 | 金额 | 说明 |
|----------|------|------|
| H股上市融资 | 31.46亿元 | 2025年12月香港主板上市 |
| 银行借款 | 184.53亿元 | 借款规模持续扩大 |
| 票据贴现融资 | 49.27亿元 | 供应链金融 |
| 永续债 | 7.00亿元 | 可递延利息的权益工具 |
| **偿还债务** | **188.42亿元** | 债务周转压力大 |

**财务风险指标：**
- **现金覆盖率**：期末现金85.68亿 / 短期借款58.27亿 = 1.47（勉强安全）
- **净债务**：有息负债约295亿 - 现金85.68亿 = 净债务约209亿
- **自由现金流**：经营现金流13.74亿 - 资本支出39.85亿 = **-26.11亿元（负值）**

## 三、赛力斯经营质量深度分析

### 3.1 经营活动现金流：✅ 强劲增长

**核心亮点：经营现金流增长28.4%至28.91亿元**

| 指标 | 数值 | 行业对比 |
|------|------|----------|
| 销售收现/营业收入 | 186.24亿/165.05亿 = 112.8% | 收现质量优秀 |
| 经营现金流净额 | 28.91亿元 | 问界品牌贡献突出 |
| 现金流/净利润 | 28.91亿/51.4亿 ≈ 0.56 | 低于1主要因折旧等非现金成本 |

**CFA视角解读：**
- **收现能力极强**：销售商品收到现金186亿元，超过营业收入165亿元，说明预收款增加或回款加速
- **华为合作效应显现**：问界品牌销售42.6万辆，同比增长10.1%，带动现金流改善
- **规模效应**：营收增长13.7%，但经营现金流增长28.4%，运营效率提升

### 3.2 投资活动现金流：✅ 资本支出理性

**投资净流出仅5.03亿元**，较2024年的16.51亿元大幅收窄：

| 投资活动明细 | 2025年 | 说明 |
|--------------|--------|------|
| 收回投资 | 68.25亿元 | 理财赎回 |
| 取得投资收益 | 0.19亿元 | 投资回报 |
| 购建固定资产 | 20.02亿元 | 产能建设 |
| 投资支付 | 27.37亿元 | 理财产品 |

**积极信号**：
- 资本支出趋于理性，产能建设进入收获期
- 理财资金灵活调配，财务配置能力较强
- **自由现金流**：28.91亿 - 20.02亿 = **+8.89亿元（正值）**，财务弹性充足

### 3.3 筹资活动现金流：✅ 融资结构优化

| 筹资活动 | 2025年 | 说明 |
|----------|--------|------|
| 吸收投资 | 18.23亿元 | 股权融资补充资金 |
| 取得借款 | 8.52亿元 | 债务融资较少 |
| 偿还债务 | 4.84亿元 | 债务压力较小 |
| 股利利息支付 | 2.22亿元 | 分红压力可控 |

**财务健康指标：**
- **现金储备充裕**：期末现金48.36亿元，同比增长664%
- **净现金状态**：现金48.36亿 > 有息负债约8.5亿，净现金约40亿
- **财务费用**：2025年上半年为-2.77亿元（利息收入>利息支出），财务成本极低

## 六、结论

**赛力斯（601127）的现金流量表表现远优于中伟新材（300919）。**

1. **赛力斯**展现了成熟的**自我造血能力**：
   - 经营现金流强劲增长28.4%
   - 自由现金流为正，财务弹性充足
   - 净现金状态，无债务压力
   - 华为合作模式成功验证，问界品牌持续贡献

2. **中伟新材**处于**高投入扩张期**，但存在隐忧：
   - 经营现金流大幅下滑65%，盈利质量下降
   - 存货激增50亿，营运资本管理承压
   - 自由现金流为负，依赖外部融资
   - 印尼项目投资回报有待验证

**投资建议**：从现金流量表角度，赛力斯的经营质量明显更优，适合作为稳健配置；中伟新材更适合作为新能源材料赛道的高风险成长标的。
