## 三、标准化数据获取Workflow

### 每日数据收集清单（按时间顺序）

#### 上午 9:15-9:30
- [ ] **USD/CNY中间价** — 中国货币网 (chinamoney.com.cn)
  - 路径: 首页 → 人民币汇率中间价
  - 确认日期为当日
- [ ] **其他货币中间价** — 欧元、日元、港币等

#### 上午 10:00-11:00
- [ ] **央行公开市场操作** — 中国人民银行官网
  - 路径: 货币政策 → 公开市场业务 → 操作公告
- [ ] **实时汇率** — 在岸/离岸人民币
  - 在岸: 东方财富、新浪财经
  - 离岸: 东方财富USD/CNH行情
  - 标注获取时间（精确到小时）

#### 下午 15:00-16:00（收盘后）
- [ ] **A股收盘数据** — 上证指数、深证成指等
- [ ] **国内期货收盘** — 碳酸锂、铜、铝等
- [ ] **外汇牌价** — 中国银行当日最新

#### 晚上/次日早上（美股收盘后）
- [ ] **美股收盘** — 道指、纳指、标普
  - 明确标注"前一交易日"因时差
- [ ] **国际大宗商品** — LME金属、布伦特原油、黄金

#### 月度数据（特定日期）
- [ ] **LPR** — 每月20日9:15发布
- [ ] **MLF** — 每月15-25日之间操作
- [ ] **动力电池数据** — 月中发布上月数据

## 四、数据质量检查清单（生成报告前必做）

### 时效性检查
- [ ] 汇率中间价是否为当日9:15发布？
- [ ] 实时汇率是否标注了获取时间？
- [ ] 央行操作是否为当日数据？
- [ ] 股市数据是否为最近收盘？

### 准确性检查
- [ ] 数字单位是否正确（元/吨、美元/盎司、基点）？
- [ ] 涨跌方向是否正确（红涨绿跌/汇率相反）？
- [ ] 涨跌幅度计算是否正确？
- [ ] 年份是否为2026年？

### 来源标注
- [ ] 每个数据是否标注了来源？
- [ ] 是否有至少2个独立来源交叉验证？
- [ ] 是否标注了数据更新时间？

6. **东方财富USD/CNH** - https://quote.eastmoney.com/forex/USDCNH.html

### 有色金属数据验证流程（新增）
**LME金属价格必须双源验证：**
1. **主要来源**: LME官方收盘价 (https://www.lme.com/) 或 Westmetall (https://www.westmetall.com/)
2. **交叉验证**: 财联社、SMM、东方财富期货行情
3. **国内现货**: 上海有色网SMM现货报价

**验证规则:**
- 如果两个来源价格差异 >$50/吨，必须查找第三个来源确认
- 标注数据来源和具体时间（收盘/盘中）
- 优先使用LME官方3-month结算价

### API/实时数据源（如可用）
- 外汇交易中心API（需申请）
- 交易所行情API
- 财经数据服务商（Wind、Choice等）

## 六、问题应急处理

### 当数据源延迟/不可用时
1. **优先使用次一级数据源**，并在报告中明确标注
2. **标注数据滞后** — "数据截至X月X日，最新数据待更新"
3. **不编造数据** — 没有数据则留白或说明"数据待发布"

### 当数据冲突时
1. 以**官方数据源**为准（央行、交易所）
2. 专业财经媒体作为交叉验证
3. 在报告中注明数据来源差异
