# 数据源可信度分级与验证规则
# 生效日期：2026-04-27
# 编制依据：用户当场指出的两起数据错误（中间价时效性+离岸价数据源）

## 一、数据源可信度分级表

| 可信度 | 等级 | 说明 | 使用规则 |
|--------|------|------|---------|
| A级 | 官方/权威机构 | 央行、中国货币网、上海黄金交易所 | 优先使用，可直接引用 |
| B级 | 国际权威数据商 | Investing.com、Westmetall、COMEX | 直接使用，建议标注 |
| C级 | 国内主流金融数据 | 东方财富、新浪财经、财联社、SMM | 直接使用，建议标注 |
| D级 | 一般财经媒体 | 证券时报、经济参考报 | 需标注，建议交叉验证 |
| E级 | 聚合/未知来源 | 我查网、实时汇率网等 | **永久禁用** |

## 二、按数据类型的数据源优先级

### USD/CNY中间价
1. **央行/中国货币网**（A级，官方发布，9:15每日）
2. **东方财富网**（C级，实时快讯）
3. **新浪财经**（C级，即时报道）

### USD/CNH离岸价
1. **Investing.com**（B级，国际权威，实时+历史数据齐全）
2. **东方财富网**（C级，逐笔交易数据）
3. **新浪财经**（C级，即时快讯）

### 在岸即期（USD/CNY）

## 三、强制验证流程

### 离岸汇率验证（新增，2026-04-27）
```
1. 首选Investing.com获取USD/CNH
2. 用东方财富网或新浪财经交叉验证
3. 两源一致 → 采用，标注双来源
4. 两源不一致 → 引入第三源（阿思达克），取多数值
5. 三源均不一致 → 标记"待确认"，报告注明数据不确定性
```

### 中间价时效验证（已有，2026-04-27强化）
```
1. 搜索"YYYY-MM-DD USD/CNY中间价"
2. 确认搜索结果中的日期=当日
3. 确认发布时间>=9:15
4. 若不满足，标记"待更新"
```

### 异常值拦截（新增，2026-04-27）
```
- 离岸与在岸价差 > 50点 → 第三源验证
- 单日涨跌 > 200点 → 多源确认
- 与昨日收盘方向相反且幅度 > 100点 → 核实是否有重大事件
```

## 四、数据源标注规范

每个数据点必须包含：
```
数值（来源，YYYY-MM-DD HH:MM）
或
数值（来源，YYYY-MM-DD）
```

**禁止的标注方式**：
- ❌ "综合"
- ❌ "网络"
- ❌ "市场"
- ❌ 无来源

**正确的标注示例**：
- ✅ 6.8221（Investing.com，2026-04-27 16:56）
- ✅ 6.8579（央行，2026-04-27 09:15）
- ✅ 19,015（财联社，2026-04-24）
