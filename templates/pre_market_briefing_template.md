# 盘前市场简报模板 v3.0（2026-04-22 更新 — 行动导向版）

## 核心原则
- **阅读时间**：5-7分钟
- **侧重点**：前瞻、预警、行动
- **目标**：快速了解一夜之间发生了什么，以及今天开盘前必须做什么
- **其余内容**：简化或删除

---

## 简报结构（4大板块）

### 头部信息
- 标题：盘前简报 — YYYY年M月D日（周X）
- 日期：动态生成
- 数据时间范围：前一日北京时间9:00至统计时点
- 一句话今日重点（统领全文的核心判断）
- 预计阅读时间标注

---

表格：

|------|----------|------|----------|
| 美股 | ... | ... | 一句话结论 |

| DXY美元指数 | ... | ... | 一句话结论 |
| 布伦特原油 | ... | ... | 一句话结论 |
| 现货黄金 | ... | ... | 一句话结论 |
| LME铜 | ... | ... | 一句话结论 |
| LME镍 | ... | ... | 一句话结论 |
| 碳酸锂 | ... | ... | 一句话结论 |
| 恒生（当日早盘） | ... | ... | 一句话结论 |

附带：隔夜要闻一句话（3-5条核心事件，每条一句话）

---

### 2. 今日风险预警
保留，但精简。

格式：
- **HIGH（红色）**：1-2条最高优先级风险，一句话描述+触发条件
- **MED（黄色）**：1-2条中等优先级风险，一句话描述+触发条件

删除：LOW级别风险、长篇情景分析

---

### 3. 今日行动清单（核心板块）
**这是整份简报最重要的部分。**

每条行动包含：
- ACTION编号 + 优先级标签
- 问题描述（1-2句）
- 今日必须做的具体步骤（2-4步，可执行）
- 决策时限

行动方向示例：
1. **确认敞口金额匹配度**（汇率影响净效应判断）
2. **监控关键变量动态**（地缘/油价/政策）
3. **启动/跟进战略评估**（技术路线/供应链）
4. **检查现有合约状态**（套保到期/展期）

---

### 4. 数据来源与时效声明
极简表格：
| 数据 | 来源 | 时间 |
|------|------|------|
| ... | ... | ... |

删除：详细的质量评级说明、交叉验证细节、免责声明长文

## 使用说明

当用户说"生成盘前简报"时：
1. 使用此v3.0模板
2. 通过 kimi_search 获取最新市场信息
3. **时间范围**：前一日北京时间9:00至统计时点
4. **输出原则**：每个数据配一句话结论，不做长篇分析
5. **行动清单必须结合"我司"敞口结构**（详见 `COMPANY.md`）—— 特别是：
   - **镍价波动**：LME镍当日价格+印尼HPM/RKAB动态
   - **USD/CNY净敞口**：中间价+在岸+离岸，分析对北美出口/香港购汇/资本层面的分别影响
   - **客户集中度**：特斯拉相关动态、订单传闻、技术路线调整
   - **钴价波动**：刚果金供应动态+LME钴价
6. 标注具体日期和数据时效
7. 输出完整简报（HTML格式，极简CSS）

## 风险仪表盘引用（盘前简报专用）

盘前简报每日监控项（从8项中选取当日变化最大的3-4项）：

| 优先级 | 监控项 | 每日必查数据 |
|--------|--------|-------------|
| P0 | **镍价波动** | LME镍收盘价、印尼HPM/RKAB新闻 |
| P0 | **USD/CNY净敞口** | 中间价(9:15)、在岸开盘、离岸走势 |
| P0 | **客户集中度（特斯拉）** | 特斯拉股价、订单传闻、技术路线公告 |
| P1 | **钴价波动** | LME钴价、刚果金地缘新闻 |
| P1 | **技术路线替代** | 铁锂/钠电产业动态（仅当有重大变化时） |
| P2 | **IDR敞口** | USD/IDR、印尼盾走势 |
| P2 | **套保执行** | 对冲覆盖率偏离目标时标注 |
| P2 | **库存跌价** | 镍钴价连续大跌时标注 |

## 数据获取优先级
1. USD/CNY中间价（央行9:15发布）- 最高优先级，**必须是当日数据**
2. LME镍/钴价格 - 最高优先级
3. 美股/原油/黄金收盘价 - 最高优先级
4. 特斯拉相关动态 - 高优先级
5. 地缘风险核心动态 - 高优先级
6. 国内宏观政策一句话 - 中优先级
7. 其他可放入"隔夜要闻一句话"
