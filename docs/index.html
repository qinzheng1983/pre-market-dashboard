2. 今日风险预警

HIGH 美伊会谈"顺利"表象下的实质风险
特朗普宣称会谈"进展顺利"，但实际是间接会谈、无面对面会晤，双方未达成公开成果。伊朗内部强硬派仍掌控海峡议题，以色列独立军事行动威胁未解除。若后续谈判无实质进展，油价可能反弹。
触发条件：伊朗宣布退出谈判 / 以色列发动空袭 / 霍尔木兹海峡通行再次受阻。
HIGH 碳酸锂快速反弹后的多空博弈加剧
电池级碳酸锂7月1日大涨至156,150元/吨（+5,300），期货主力合约+6.13%至163,360。中矿资源宣布停产检修（7月底结束），JXW据传复产。但7月仓单压力仍处高位（4.87万吨），短期反弹后可能面临交割压力。
触发条件：仓单强制注销前现货抛压 / 复产消息确认 / 需求端不及预期。
MED 美股科技股高位回调传导
美股7月1日纳指-0.66%，芯片板块领跌。A股7月1日算力、半导体已现机构大规模调仓。若美股科技持续回调，可能影响港股及A股科技板块情绪。
触发条件：纳指连续两天下跌超1% / 费城半导体指数大幅回调。
