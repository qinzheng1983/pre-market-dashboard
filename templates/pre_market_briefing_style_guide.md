# 盘前市场简报 HTML 模板规范
# 用户确认版本：2026-03-31

## 配色方案
- 主色调: #1a1a2e (深蓝) — 标题栏、表格表头
- 强调色: #e94560 (红) — USD/CNY高亮、风险标签、上涨标识
- 上涨: #e94560 (红色)
- 下跌: #27ae60 (绿色) — 人民币升值用绿色
- 警示背景: #ffe6e6 (浅红) — 高度警惕区域
- 提醒背景: #fff9e6 (浅黄) — 关注区域

## 排版规范
1. 标题栏: 纯色背景(禁用渐变)，白色文字
2. 区块分隔: 左侧3px红色边框线
3. 表格: 标准HTML表格，表头深蓝底白字
4. 标签: 纯色背景标签(.risk-high / .risk-med)

## 禁用元素
- 渐变背景 (linear-gradient)
- emoji字符
- 复杂CSS动画
- flexbox布局 (部分邮件客户端不支持)

## 兼容性要求
- 139邮箱仅支持基础HTML和CSS
- 使用简单表格布局
- 确保移动端正常显示
