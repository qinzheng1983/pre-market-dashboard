1.2 因子模型断裂点诊断（自我挑战）

每个因子列出：① 模型假设 ② 反逻辑挑战 ③ 历史失效案例 ④ 重新定价节点。当多个因子同时断裂时，模型输出不可信。

部分修复：利差因子假设：央行加息 → 利率优势 → 资本回流 → 本币升值

反逻辑挑战BI 5/20加息50bp至5.25%，6/9再加25bp至5.50%，6/18再加25bp至5.75%，利率大幅反超Fed（3.50-3.75%），但IDR在6月9日盘中仍触及18,222。这不是"利差不够"，而是"利差修复需要时间"。当5Y CDS从90bp跳升至97bp（6/8），主权风险溢价上升一度完全吞噬了利差优势。但6月10-11日外资流入SRBI/SBN 19.02万亿盾显示，当利差足够大时，部分资本开始回流。关键问题：这种回流是"趋势逆转"还是"短暂套利"？
历史失效2013年Taper Tantrum：BI加息175bp，IDR仍贬值20%。当主权CDS>100bp时，利差因子对EM货币完全失效。当前97bp已极度逼近该阈值。但BI此次累计加息100bp（从4.75%→5.75%）且外资已开始流入，与2013年"加息无效"有所不同。
重新定价节点如果BI维持5.75%或继续加息到6.0%，且同时CDS回落至85bp以下，市场可能重新定价利差。仅加息不收敛CDS = 效果有限。量化阈值：CDS < 85bp AND BI Rate > Fed Upper Bound + 2.0%。当前5.75% vs 3.75% = +2.0%，刚好触及阈值。
当前 verdict部分修复中。利差因子权重从0%回升至约15-20%。BI累计加息100bp后，外资首次出现实质性回流（19.02万亿盾），这是积极信号。但股市年内跌36%、外资撤出33亿美元显示"回流"规模远不足以逆转趋势。利差因子尚未恢复为"主导因子"，但也不再是"完全断裂"。

脆弱：经常账户韧性假设：连续顺差 → 基本面支撑本币

反逻辑挑战1-4月累计顺差56.4亿（月均14.1亿）是"存量"，但进口增速13.40% vs 出口增速5.48%是"流量"。市场看的是二阶导数：顺差在收窄。4月单月顺差仅9亿（BPS），已低于1月9.5亿和2月12.7亿。中国逆差24.7亿（1月）是最大来源——如果中国镍需求放缓（Danantara出口集中化），顺差可能快速转负。油气逆差85.2亿（1-4月）完全吞噬了非油气顺差141.6亿中的60%。

新增风险：外汇储备5月底1,449亿美元，连续5个月下降，自2025年12月累计减少116亿美元。央行干预汇市消耗外储的速度（月均约23亿）正在侵蚀"连续顺差"的正面效应。
历史失效印尼2011年经常账户盈余+0.2%GDP，但2013年转为-3.2% deficit，IDR同期贬值26%。"连续顺差"到"突然逆差"的切换，EM市场通常只给1-2个季度预警。当前月均顺差已从2025年的34.2亿（全年410.5亿/12）降至14.1亿，速度惊人。外汇储备消耗速度更快。
重新定价节点如果BPS 5-6月数据确认顺差收窄至<10亿/月或转负，市场将从"缓慢贬值"切换到"加速贬值"模式。量化阈值：月度顺差 < 5亿美元连续2个月，或单个月度逆差。外储方面，如果降至1,400亿以下（约2个月），市场可能恐慌。
当前 verdict脆弱中。经常账户因子权重从传统的40%降至约15%，且呈衰减趋势。"顺差存在"不等于"本币有支撑"——流量恶化+资源政策不确定性+外储消耗已严重折扣了存量。

主导：风险溢价假设：CDS↑ + 资本外流 + 政策不确定性 → 本币贬值压力

反逻辑挑战CDS 97bp是"高"，但相对历史危机水平（1998 >500bp, 2013 >200bp, 2020疫情>180bp）仍是"中等偏高"。市场定价的是"走向危机的路径"，而非"危机已发生"。这创造了一个微妙的反逻辑：如果风险事件兑现（Danantara执行混乱/镍矿配额争议升级），CDS会跳升突破100bp；但如果政策执行平稳，CDS从97bp回落的空间也很大——意味着IDR反弹的弹性被低估。

新增反逻辑：BI累计加息100bp后，6月10-11日外资流入19.02万亿盾，显示"高利率"开始吸引部分套利资本。但股市年内跌36%、外资撤出33亿美元显示"套利流入"远小于"权益流出"。风险溢价因子的"反逻辑"在于：它可能被过度定价，也可能被定价不足——在制度变革期，两者概率都不低。
历史失效风险溢价因子在EM中极少"失效"——它通常是最晚失效的因子。但存在反例：2016年印尼CDS从150bp回落至80bp，IDR同期升值12%，因为Jokowi改革红利兑现。风险溢价因子的失效条件是：出现一个被市场认可的制度性改善信号。
重新定价节点上行（更差）：CDS > 100bp触发量化基金止损，>150bp触发评级下调担忧（当前Moody's Baa3，仅一步之遥到垃圾级）。下行（更好）：Danantara执行透明化 + 贸易顺差恢复>15亿/月 + CDS回落至<80bp + 外资持续回流。量化阈值：CDS 100bp为恐慌加速点，80bp为情绪修复点。
当前 verdict主导中，权重约65-70%。风险溢价仍是当前被市场定价的最重要因子，但6月BI累计加息100bp后权重已从85%略有下降。Danantara 6/1启动是"已知的未知"——执行细节将决定CDS是突破100bp还是回落。

跨因子断裂：USD/IDR vs USD/CNY 相关性假设

模型假设用USD/CNY远期间接对冲IDR风险，假设USD/IDR和USD/CNY正相关、同向波动。
反逻辑挑战压力测试：USD/IDR +20%（→21,500），USD/CNY +2%（→6.957）→ IDR/CNY贬值25.7%。历史正相关在此场景完全破裂。为什么？因为USD对IDR的升值被印尼特有的风险溢价放大（资源国有化、镍矿配额-30%），而USD对CNY的升值被中国央行的逆周期因子抑制。两个USD不是同一个USD。

6月新验证：6月9日USD/IDR盘中触及18,222，但USD/CNY中间价仅小幅波动。USD/IDR的波动幅度（±2-3%）远超USD/CNY（±0.3%），相关性在极端场景下继续弱化。
重新定价节点如果USD/CNY中间价突破7.0，中国央行可能启动逆周期因子或宏观审慎参数调整，这将进一步拉大USD/IDR和USD/CNY的分化。量化阈值：USD/CNY中间价 > 7.0。
当前 verdict相关性在极端场景下不可靠。任何基于"间接对冲"的敞口管理，必须单独建模USD/IDR尾部风险。

第三层：USD/IDR 情景分析（Scenario Analysis）

以下概率不是模型输出，是基于第一层断裂诊断后的主观判断：风险溢价因子当前权重65-70%，粘性仍高但边际下降；经常账户因子权重15%，呈衰减趋势；利差因子权重15-20%，边际回升。所有情景直接以 USD/IDR 为观察标的，不再假设 IDR/CNY 的传导关系。

概率 15%（上调）
乐观情景：改革红利释放，风险溢价因子回落
触发条件Danantara执行透明化 + 镍矿配额实际执行未超预期收紧 + 中东缓和 + 中国镍需求持续 + 贸易顺差恢复>15亿/月 + CDS回落至<80bp + 美联储转鹰预期被市场充分消化
USD/IDR从当前17,875回落，目标区间16,900-17,300（年底），较当前贬值压力释放约3-5.5%
对中伟印尼基地本地成本（IDR计价）折算USD下降，利润率改善。MHP出口USD收入不变，但IDR成本端减负。但注意：即使回到17,100，仍较2025年底约16,300贬值约4.9%——这不是"升值"，只是"贬值放缓"。
为什么概率上调BI累计加息100bp后出现外资实质性回流（19.02万亿盾），利差因子从"完全断裂"修复为"部分有效"。如果BI维持5.75%且美联储转鹰被市场消化，利差优势可能重新吸引资本。但概率仅15%，因为Danantara 6/1刚启动，资源政策不确定性已造成实际冲击（魏桥码头被查封），且外储连续5个月下降。
概率 50%（维持）
基准情景：风险溢价维持高位，USD/IDR 缓慢贬值
触发条件Fed维持利率不变或小幅加息（点阵图3.8%中值），印尼经常账户温和恶化，央行持续干预但效果有限（外储消耗至1,400-1,450亿），Danantara分阶段实施不引发市场恐慌但持续压制情绪，CDS维持90-105bp区间

4.1 期权波动率曲面：25d 与 10d Risk Reversal（Call vs Put 哪个更贵）

在 USD/IDR 期权市场，Call = 买入USD / 卖出IDR（押注USD升值，IDR贬值）；Put = 卖出USD / 买入IDR（押注IDR反弹）。

Risk Reversal（RR）定义为：RR = IV(25d Call) − IV(25d Put)。RR > 0 表示 Call 比 Put 贵 → 市场更怕IDR贬值；RR < 0 表示 Put 比 Call 贵 → 市场更怕IDR反弹。

指标 估算读数 解读 交易含义
25d Risk Reversal +3.5 ~ +5.0 vol pts Call 显著贵于 Put。25-delta Call（ mildly OTM USD看涨）隐含波动率比同delta Put高约3.5-5.0个百分点。 市场主流押注IDR继续贬值，对冲需求集中在USD Call。注意：6月BI累计加息100bp后，RR较6月初的+4.0~+5.5有所收窄，显示加息部分缓解了贬值预期，但仍处于高位。
10d Risk Reversal +5.5 ~ +7.5 vol pts 10d Call 比 10d Put 更贵，尾部风险定价（deep OTM）显示市场对IDR暴跌仍有额外恐慌，但较6月初的+6.0~+8.0略有收窄。 尾部风险溢价仍高但边际下降。买入10d Put（IDR反弹保险）相对"便宜"，但流动性差；卖出10d Call（USD极端升值）可收取高权利金，但风险极大。
10d vs 25d 差距 +2.0 ~ +2.5 vol pts 10d RR 比 25d RR 更正值，说明越远离平价的USD Call越贵。波动率微笑呈现陡峭的右偏（right skew），但较6月初边际收敛。 "买25d Call + 卖10d Call"的比率价差可能捕捉skew收敛，但需承担gamma风险。BI加息后skew边际收敛，策略风险较月初降低。
Skew 分析结论：
· 当前 USD/IDR 期权市场呈现右偏：Call（USD看涨/IDR看跌）全面贵于 Put，但较6月初边际收敛
· BI累计加息100bp后，RR从+4.0~+5.5收窄至+3.5~+5.0，说明加息部分缓解了贬值预期——市场认为加息是"有效防御"而非"被动应对"
· 历史对比：2013年Taper Tantrum期间，USD/IDR 25d RR 峰值为 +6.2 vol pts；2018年土耳其危机传染期峰值为 +8.5 vol pts。当前+3.5~+5.0处于"高但未到极端"区间，边际改善
· 关键技术信号：如果10d/25d spread突破+3.0，说明恐慌进入加速期；如果收窄至<+1.5，说明贬值预期被消化，可能临近短期底部。当前+2.0~+2.5处于"中间地带"
