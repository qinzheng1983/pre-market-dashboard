# 财资日报数据源优先级与验证规则

## USD/CNY中间价（最高优先级）

**铁律**：ONLY认央行授权公告

| 优先级 | 来源 | URL | 说明 |
|--------|------|-----|------|
| 1 | 中国货币网 | chinamoney.com.cn/bkccpr/ | 央行授权发布平台 |
| 2 | 央行官网 | pbc.gov.cn | 央行公告原文 |
| 3 | 新浪财经 | finance.sina.com.cn | 转载央行公告 |
| 4 | 财联社 | cls.cn | 转载央行公告 |
| ❌ 禁用 | chl.cn / 汇率网 / 查汇率 | — | 聚合站数据延迟，日期标注混乱 |

**校验流程**：
1. 搜索"YYYY年MM月DD日 中国外汇交易中心 人民币汇率中间价公告"
2. 确认公告日期 = 报告日期（关键！chl.cn常把前日数据标为"今日"）
3. 提取中间价后，用 chinamoney.com.cn/bkccpr/ 二次确认
4. 标注来源时写：央行授权公告 + 新浪财经/财联社
