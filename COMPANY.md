## 四、区域市场敞口矩阵（四大市场）

### ① USD/CNY — 北美市场

| 维度 | 详情 |
|------|------|
| **核心客户** | **特斯拉**（最重要的国际战略客户）、其他北美车企 |
| **产品偏好** | 高镍三元前驱体（高镍低钴化路线） |
| **贸易条款** | FOB / CIF |
| **结算货币** | **美元**（USD） |
| **信用周期** | 60-90天 |
| **敞口方向** | **美元正敞口**（Long USD）— 出口应收美元，USD升值→折算人民币收入增加 |
| **合规路径** | 通过印尼等"友好国家"产能布局和供应链重构，寻求符合IRA等法规的原产地要求 |
| **⚠️ 风险点** | 若印尼失去"友好国家"资格，北美市场准入可能丧失 |

### ② EUR/CNY — 欧洲市场

| 维度 | 详情 |
|------|------|
| **生产基地** | 德国、摩洛哥（本地化生产基地） |
| **结算货币** | **欧元**（EUR）为主，部分合同采用**美元计价、欧元结算**的混合模式 |
| **敞口方向** | **欧元/美元混合敞口** — 美元计价合同为USD/CNY敞口，欧元结算合同为EUR/CNY敞口 |
| **⚠️ 复杂性** | "美元计价、欧元结算"模式意味着合同金额锁定美元，但实际支付用欧元——存在USD/EUR交叉汇率风险 |
| **合规压力** | 欧洲电池法规（EU Battery Regulation）、CBAM碳边境税 |

### ③ KRW/CNY — 日韩市场

| 维度 | 详情 |
|------|------|
| **核心客户** | 三星SDI、LG化学等战略客户 |
| **生产模式** | 韩国设有材料基地，从"出口贸易"升级为"**本地生产、本地服务**" |
| **结算货币** | **美元 + 韩元**为主，部分日本客户采用**日元**结算 |
| **敞口格局** | **多币种并行**：USD应收 + KRW应收 + JPY应收 |
| **供应链优势** | 本地生产缩短响应周期，降低物流成本和汇率暴露 |
| **⚠️ 监控重点** | KRW/CNY波动直接影响韩国基地利润折算，JPY波动影响日本客户应收 |

### ④ IDR/CNY — 东南亚/印尼市场

| 维度 | 详情 |
|------|------|
| **双重属性** | **资源基地**（全球最大镍资源供应基地）+ **制造与贸易枢纽** |
| **产品流向** | ①出口至韩国、日本等第三国；②运回中国总部深加工；③印尼本地及周边销售 |
| **结算货币** | **美元 + 印尼盾 + 人民币**多币种并行 |
| **内部贸易** | 复杂的集团内部贸易网络涉及多币种结算 |
| **敞口特征** | **USD/IDR + IDR/CNY + USD/CNY 三重交叉敞口** |
| **⚠️ 关键变量** | IDR贬值→印尼本地成本上升；USD/IDR波动→内部贸易定价重置 |

### 5.1 敞口总览图

```
┌─────────────────────────────────────────────────────────────┐
│                    中伟新材多币种敞口全景                      │
├─────────────────────────────────────────────────────────────┤
│  ① USD/CNY 北美出口：美元应收（Long USD）                     │
│  ② USD/CNY 香港购汇：美元支出（Short USD）← 印尼子公司支付      │
│  ③ USD/CNY 资本层面：子公司美元负债（Long USD）                 │
│  ④ EUR/CNY 欧洲销售：欧元应收（Long EUR）+ 美元计价合同          │
│  ⑤ KRW/CNY 韩国基地：韩元应收/应付 + 美元合同                  │
│  ⑥ JPY/CNY 日本客户：日元应收（Long JPY）                     │
│  ⑦ IDR/CNY 印尼本地：印尼盾收支 + 内部贸易                     │
│  ⑧ USD/IDR 交叉汇率：印尼基地内部定价基准                      │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 各区域敞口方向与影响

| 市场 | 敞口方向 | 汇率影响 | 有利情景 | 不利情景 |
|------|---------|---------|---------|---------|
| **北美出口** | Long USD（美元应收） | USD/CNY升值→人民币收入增加 | 美元升值 | 人民币升值 |
| **香港购汇** | Short USD（美元支出） | USD/CNY贬值→购汇成本降低 | 人民币升值 | 美元升值 |
| **资本层面** | Long USD（子公司美元负债） | USD/CNY升值→负债折算减少 | 美元升值 | 人民币升值 |
| **欧洲销售** | Long EUR + USD计价合同 | EUR/CNY升值→收入增加；USD升值→合同金额增加 | 欧元/美元升值 | 欧元/美元贬值 |
| **韩国基地** | Long KRW + USD合同 | KRW/CNY升值→利润折算增加 | 韩元升值 | 韩元贬值 |
| **日本客户** | Long JPY（日元应收） | JPY/CNY升值→收入增加 | 日元升值 | 日元贬值 |
| **印尼本地** | USD/IDR + IDR/CNY交叉 | IDR贬值→本地成本上升 | IDR稳定 | IDR剧烈贬值 |

### 5.3 净敞口效应分析（关键判断）

**核心矛盾**：
- **北美出口**（美元正敞口）与**香港购汇**（美元负敞口）存在部分自然对冲
- **资本层面**（美元正敞口）与**贸易层面**（美元负敞口）存在部分自然对冲
- **但**：欧洲（EUR）、日韩（KRW/JPY）、印尼（IDR）的敞口**无法被USD/CNY自然对冲**

**净敞口判断逻辑**：
1. USD/CNY走势对北美出口 + 香港购汇 + 资本层面的**净效应**取决于金额匹配度
2. EUR/CNY、KRW/CNY、JPY/CNY、IDR/CNY的敞口**各自独立**，需要分别评估
3. **2025年汇兑损益激增45%说明**：当前对冲体系未能覆盖多币种敞口的叠加效应

## 六、对冲策略体系（多币种动态框架）

### 6.1 对冲优先级矩阵

| 币种对 | 敞口规模 | 波动率 | 对冲优先级 | 推荐工具 |
|--------|---------|--------|-----------|---------|
| **USD/CNY** | 最大 | 中 | ⭐⭐⭐⭐⭐ | 远期、期权、领子 |
| **EUR/CNY** | 大 | 中高 | ⭐⭐⭐⭐⭐ | 远期、期权 |
| **KRW/CNY** | 中 | 中高 | ⭐⭐⭐⭐ | 远期、货币互换 |
| **IDR/CNY** | 中 | 高 | ⭐⭐⭐⭐ | 远期（流动性受限） |
| **JPY/CNY** | 小 | 中 | ⭐⭐⭐ | 远期 |
| **USD/IDR** | 内部贸易 | 高 | ⭐⭐⭐⭐ | 内部定价对冲、自然对冲 |

### 6.2 情景化对冲矩阵（USD/CNY主敞口）

| 情景 | USD/CNY走势 | 北美出口影响 | 香港购汇影响 | 资本层面影响 | 净效应 | 对冲建议 |
|------|------------|------------|------------|------------|--------|---------|
| 人民币升值 | 6.8 → 6.5 | 不利（收入减少） | 有利（购汇成本低） | 不利（负债膨胀） | 取决于金额匹配 | 重点保护资本+北美出口 |
| 美元升值 | 6.8 → 7.2 | 有利（收入增加） | 不利（购汇成本高） | 有利（负债减少） | 取决于金额匹配 | 重点保护香港购汇 |
| 剧烈波动 | 大幅双向波动 | 两边交替承压 | 两边交替受益 | 两边交替受益 | 不确定性高 | 两端分别对冲 |

### 6.3 期权策略优先级（多币种适配）

| 期权策略 | 适用币种 | 适用敞口 | 适用场景 | 对我司优先级 |
|---------|---------|---------|---------|-------------|
| 买入看跌期权 (Protective Put) | USD/CNY | 资本层面+北美出口 | 预期人民币升值，保护子公司负债+出口收入 | ⭐⭐⭐⭐⭐ |
| 买入看涨期权 | USD/CNY | 香港购汇 | 预期美元升值，锁定购汇成本 | ⭐⭐⭐⭐ |
| 领子期权 (Collar) | USD/CNY/EUR/CNY | 两端均可 | 区间波动预期，零成本锁定 | ⭐⭐⭐⭐ |
| 海鸥期权 (Seagull) | USD/CNY | 资本层面 | 温和贬值预期，降低成本 | ⭐⭐⭐ |
| 比率远期 (Ratio Forward) | USD/CNY/EUR/CNY | 大额敞口 | 优于远期汇率，接受非对称 | ⭐⭐⭐⭐ |
| 累计期权 (Accumulator) | 多币种 | 大额渐进对冲 | 逐日结算，逐步建仓 | ⭐⭐（高风险，慎用） |

### 6.4 对冲执行规则
- **每次财资日报必须结合敞口结构分析汇率影响**
- 汇率板块不能只做泛泛的市场评论，必须回答：
  - "今天USD/CNY走势对我司北美出口/香港购汇/资本层面分别意味着什么"
  - "今天EUR/CNY、KRW/CNY、IDR/CNY走势对我司欧洲/日韩/印尼业务分别意味着什么"
- 对冲建议必须区分：
  - "保护出口收入"（北美、欧洲、日韩）
  - "保护购汇成本"（香港支付）
  - "保护资本负债"（子公司借款）
  - "保护本地利润折算"（韩国基地、印尼本地）
- **2025年汇兑损益45%激增是底线数据**：对冲目标是将年度汇兑损益波动控制在±20%以内

## 七、行业与供应链特征

### 7.1 原材料依赖

| 原材料 | 价格驱动 | 关键变量 | 对我司影响 |
|--------|---------|---------|-----------|
| 硫酸镍 | LME镍价 + 印尼HPM政策 | RKAB配额、出口限制 | 直接影响前驱体生产成本 |
| 硫酸钴 | 钴价（刚果金为主） | 地缘政治、库存周期 | 三元材料成本核心变量 |
| 磷酸铁锂 | 锂价 + 磷矿自给率 | 碳酸锂波动、磷矿投产进度 | 技术路线切换影响产能利用率 |

### 7.2 印尼基地产品流向与内部贸易网络

```
印尼基地（镍钴中间品）
    │
    ├─→ 出口韩国、日本（第三国销售）— USD结算
    ├─→ 运回中国总部深加工 — USD/CNY/IDR多币种内部结算
    └─→ 印尼本地及周边市场销售 — IDR/USD结算
```

**⚠️ 内部贸易定价风险**：
- 印尼→中国总部的转移定价涉及USD/IDR→USD/CNY→IDR/CNY三重汇率
- USD/IDR波动直接影响内部成本核算
- 需建立动态内部定价调整机制

### 7.3 印尼政策敏感度（核心成本变量）
- **HPM新政**：印尼矿产和煤炭部长发布的矿产基准价格，直接影响镍矿采购成本
- **RKAB配额**：矿产开采配额调整，影响原料供应稳定性
- **出口限制**：镍矿出口禁令或限制措施，直接影响硫酸镍供应链
- **⚠️ 仪表盘必须保留"印尼政策"项**：这是中伟新材区别于其他企业的核心风险变量

### 7.4 技术路线风险
- **三元 → 铁锂迁移**：行业从"三元为王"转向"铁锂主导"，影响前驱体产能利用率
- **钠离子电池替代**：对锂电池体系的潜在替代风险
- **固态电池**：公司战略重点，抢占下一代技术制高点
- **磷矿自给**：磷矿项目（9844万吨资源量）投入节奏与现金流匹配
